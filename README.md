Un README corto explicando:

# ¿Qué hiciste? 

Hice una calculadora de rutas 
Calcula el camino "optimo" entre dos puntos
Y uso la herramienta Tkinter para que sea mas visual la ejecucion del algoritmo 

# ¿Qué algoritmo usaste?

Use el algoritmo A*

# ¿Qué aprendiste?
Aprendi sobre varios algoritmos de busqueda, y las diferencias entre ellos (BFS, Dikstra, A*)

Pude ver como A* es utilizado en muchas areas, como videojuegos, Google maps etc.


Al principio, intente resolver este desafio usando JavaScript pero cuando vi que estaba tardando demasiado tiempo en aprender todo lo necesario (HTML, CSS, JavaScript) decidi volcer a lo conocido (Python) por que tampoco sabia sobre la herramienta Tkinter

Aprendi un poco sobre la herramienta Tkinter y asi pude terminar este desafio
